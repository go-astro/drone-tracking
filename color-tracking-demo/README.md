# color-tracking-demo

## Description:
A demo to fool-around with and show color tracking possibilities

## Requires:
    CMake >= 2.8
    OpenCV >= 2.4 (can also be a own-built version, edit the CMakeLists.txt accordingly)

## Install:
    mkdir build
    cd build
    cmake ..
    make

## Run:
    ./color-tracking-demo
